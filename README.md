# Getting Started

Open [http://localhost:3000](http://localhost:3000)

## Available Scripts

In the project directory, you can run:

### Docker
```
docker compose up
```

### NPM Scripts

Development Server
```
npm run start 
```
Production Build
```
npm install
npm run build
npx serve build 
```
